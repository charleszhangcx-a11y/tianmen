# 天门方言语音包说明

把录好的 mp3 文件丢进这个目录(`public/assets/audio/`)即可,文件名要和下面对应。
**没有 mp3 时游戏照常运行**,会用屏幕上的文字气泡兜底(可在 `public/game.js` 的 `VOICE_TEXT` 里改成天门方言)。

## 需要准备的文件

| 文件名 | 触发时机 | 建议内容(天门方言搞笑版,自己发挥) |
|--------|----------|-----------------------------------|
| `play.mp3` | 任意出牌 | 「出哒!」「看老子的!」 |
| `bomb.mp3` | 出炸弹 | 「炸死你个板马!」 |
| `pass.mp3` | 要不起/摸牌 | 「要不起咧~」「摸一张克」 |
| `win.mp3` | 出完牌(赢) | 「我跑脱哒!」「头游是我滴!」 |

## 怎么录

1. 用手机录音 App 录,或找天门本地朋友录,口味越足越好。
2. 导出成 mp3(其他格式可用在线工具转 mp3)。
3. 文件名严格按上表命名,放进本目录。
4. 重新部署(或本地重启),进游戏点「🔊 语音」开启即可听到。

## 想加更多语音(比如不同牌型不同台词)

打开 `public/game.js`,找到 `VOICE_MAP`:

```js
const VOICE_MAP = {
  bomb: "bomb.mp3",
  single: "play.mp3",   // 想给单张单独配音,就改成 "single.mp3"
  pair: "play.mp3",     // 对子单独配音 -> "pair.mp3"
  ...
};
```

把对应牌型改成新文件名,再把新 mp3 放进来即可。

## 注意(微信里的限制)

微信内置浏览器有时要求**用户先点一下页面**才允许播放声音。本游戏靠玩家点「出牌/开始」等按钮触发,通常没问题;若第一声没响,点一下「🔊 语音」按钮即可解锁。
